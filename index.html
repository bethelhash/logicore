    Methodology & Data Sources

    Freight Class (NMFC): Determined from cargo density (lbs/ft³) using the 17-band NMFC 100 Series density-to-class table. Classes 50–500.

    Base Freight Rates: Indicative ranges from US Surface Transportation Board (STB) filings for LTL/FTL, IATA tariff guides for air freight, and World Container Index (WCI) for ocean. Actual rates vary significantly by contract, lane, and volume.

    Import Duties: WTO Most Favoured Nation (MFN) schedules via USITC HTS database. HS code prefixes override to sector-specific rates (electronics, steel, textiles).

    CBAM Carbon Fee: EU Reg. 2023/956 default emission factors per commodity × current EU ETS price. Applies to iron/steel, aluminium, cement, fertilisers, hydrogen, and electricity to EU.

    Transport CO₂e: GLEC Framework v3 emission factors per mode (kg CO₂e per tonne-km) × approximate lane distance.

    3D Load Planner: ISO 668 internal container dimensions. Weight-first bin-packing heuristic. Results illustrative only.
